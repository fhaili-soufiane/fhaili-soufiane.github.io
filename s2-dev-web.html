Semestre 2

Technologies et développement web

Les fondamentaux du web — HTML, CSS, JavaScript — pour concevoir des ressources pédagogiques en ligne.

Ressources de cours

Ce module introduit les technologies du web : HTML pour la structure, CSS pour la présentation et JavaScript pour l'interactivité. L'objectif est de maîtriser les bases de la création de pages web, compétence utile pour concevoir des supports pédagogiques en ligne, des exerciseurs interactifs ou — comme ce portfolio — un espace de présentation de son travail.

Ce que j'ai retenu : Le web repose sur trois piliers complémentaires : HTML (le fond), CSS (la forme) et JavaScript (le mouvement). Chaque technologie a un rôle précis et ils fonctionnent ensemble. Un site web bien conçu est accessible, responsive et léger. Ce portfolio est la mise en pratique de ce que j'ai appris dans ce module.

Compétences développées

HTML5

Structure sémantique, balises, formulaires, multimédia. Accessibilité et bonnes pratiques.

CSS3

Sélecteurs, box model, flexbox, grid. Responsive design, animations, variables CSS.

JavaScript

DOM, événements, fetch. Interactivité et dynamisme côté client.

GitHub Pages

Hébergement statique gratuit. Déploiement continu depuis un dépôt Git.

Bonnes pratiques

Performance, accessibilité (a11y), SEO de base. Navigation clavier, contrastes.

Outils de développement

Inspecteur du navigateur, éditeur de code, validation W3C.

Réflexion personnelle

Ce portfolio est le fruit de ce module. Chaque ligne de HTML, chaque règle CSS, chaque fonction JavaScript a été écrite à la main, sans framework ni générateur. C'est un choix délibéré : pour enseigner le web à des collégiens, il faut maîtriser les fondamentaux. Ce projet m'a permis de consolider mes compétences techniques tout en créant un outil utile pour ma soutenance.
