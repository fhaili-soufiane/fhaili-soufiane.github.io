Semestre 1

Bases de données

Introduction aux bases de données, au modèle relationnel, au langage SQL et à la conception de schémas avec le modèle Entité-Association.

Ressources de cours

Ce module, réalisé en collaboration, couvre l'ensemble des fondamentaux des bases de données : de la différence entre donnée et information jusqu'aux requêtes SQL complexes. Chaque étudiant a travaillé sur un thème spécifique, produisant des fiches synthétiques qui couvrent l'intégralité du programme.

Ce que j'ai retenu : Une base de données n'est pas qu'une collection de données — c'est un système organisé qui garantit l'intégrité, la cohérence et l'efficacité des accès. Le modèle relationnel et le langage SQL sont les piliers incontournables de la gestion de données moderne.

Réflexion personnelle

Pourquoi les bases de données sont essentielles ?

Dans un monde numérique où les données sont partout, savoir concevoir, interroger et gérer une base de données est une compétence fondamentale. Que ce soit pour une application web, un système de gestion scolaire ou une application mobile, les principes appris dans ce module — du modèle conceptuel aux requêtes SQL — sont universels.
