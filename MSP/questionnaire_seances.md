### Contexte

### Déroulement
Remplis ce tableau :

| Phase | Durée | Ce que tu as fait | Ce que les élèves ont fait |
|---|---|---|---|
| Accroche | | | |
| ... | | | |
| ... | | | |

### Auto-évaluation par axe (3-4 lignes chacun)

**Didactique** :
- Les élèves ont-ils compris ? Comment l'as-tu vérifié ?
- Y a-t-il eu des erreurs / confusions ? Comment les as-tu gérées ?
- As-tu atteint ton objectif ?

**Pédagogique** :
- As-tu respecté ton timing ? Qu'est-ce qui a pris plus/moins de temps ?
- Tes supports étaient-ils adaptés ?
- Comment as-tu organisé le tableau / la salle ?

**Relationnel** :
- Comment était le climat de classe ?
- As-tu réussi à capter l'attention ?
- Y a-t-il eu des incidents ? Comment les as-tu gérés ?
- Le tuteur t'a-t-il fait un retour ?

### Pistes d'amélioration
- Qu'as-tu trouvé réussi ?
- Que referais-tu différemment la prochaine fois ?

### Bilan réflexif
- Qu'as-tu appris de cette expérience ?
- Quel est ton objectif pour la prochaine séance ?
